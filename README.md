# ClientPortal — Project Management SaaS

A full-stack MERN web application for managing client projects and tasks.

## 🔗 Live Demo
[https://client-portal-inky-three.vercel.app/](https://client-portal-inky-three.vercel.app/)

## ✨ Features
- JWT Authentication (Register / Login)
- Role-based access (Admin / Client)
- Create and manage projects
- Kanban task board (To Do / In Progress / Done)
- Task priority and due dates
- Fully responsive UI

## 🛠️ Tech Stack
- **Frontend:** React, Vite, Tailwind CSS, React Router
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas
- **Auth:** JWT + bcrypt
- **Deployment:** Vercel (frontend) + Render (backend)

## 🚀 Run Locally

### Backend
cd server
npm install
npm run dev

### Frontend
cd client
npm install
npm run dev
